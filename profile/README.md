# 🚀 LeMMong

### The World’s #1 Memory Layer for RAG & AI Agents

我们致力于为大模型提供最强的大脑“记忆中心”。不仅是存储，更是精准、多模态的实时调取。

---

## 💡 核心能力

- **多模态记忆检索**：完美支持 **Markdown, PDF, 音频, 视频**。无论信息藏在文档的哪一页，还是视频的哪一秒，我们都能找出来。
- **原子级精准回溯**：拒绝幻觉！大模型的每一句回答都能**精准链接**到原始资料的对应片段。
- **Agent 专属长短期记忆**：为 AI Agent 提供持久化记忆，让它像人类一样记住对话背景与复杂知识。
